# Example Specification

## Purpose

This is an example specification demonstrating the delta-spec format. Delete this file when you create your first real spec.

## Requirements

### Requirement: Example Feature
The system SHALL demonstrate the specification format.

#### Scenario: User reads example
- GIVEN a new user of delta-spec
- WHEN they read this example specification
- THEN they understand the format
- AND they can create their own specs

### Requirement: Scenario Format
The system SHOULD use Given/When/Then format for scenarios.

#### Scenario: Complete scenario
- GIVEN a requirement that needs testing
- WHEN a scenario is written
- THEN it includes GIVEN (precondition)
- AND it includes WHEN (action)
- AND it includes THEN (outcome)
